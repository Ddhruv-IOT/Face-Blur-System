# Face Blur System

## ℹ️ℹ️ Overview
A simple python code to detect a human face and blur it.
<br/> It has been made using OpenCV.

## ⚒️⚒️ Tools and Technologies Used:

### 💿💿 Python Modules used:
- OpenCV for detecting face and blurring it.

### 💿💿 Softwares Used 
- Spyder 5.0.5
- CMD
- Git Bash 

### 💽💽 OS Used:
- Windows 10 

## ✔️✔️ Features  
- Detect the human face and blur it.
